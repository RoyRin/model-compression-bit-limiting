# Lossy Compression Methods

## Approaches

**1. Best-of-N Selection**
- Generate N outputs, pick the one that compresses best

**2. Question-Answer Guidance** 
- LLM asks guiding questions to help SLM improve its answer iteratively

**3. Selection Generation**
- Generate multiple SLM candidates, LLM picks best continuation 

**4. Direct Compression**
- Compress text using arithmetic coding with language models
